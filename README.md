## Description
This project provides a simple interest calculator designed to help users quickly compute total interest earned or paid over a specified period.

## Formula
The simple interest formula used in this project is:

Simple Interest (SI) = (P * R * T) / 100

Where:
- *P* = Principal amount (Initial sum of money)
- *R* = Annual interest rate (in percentage)
- *T* = Time period (in years)

## Features
- Fast and accurate simple interest calculation
- Easy-to-understand inputs for principal, rate, and time
- Clear output display showing the final calculated interest and total amount
